## 🔐 Fonctionnalités

- **Authentification sécurisée** : Admin (`5710`) et invité (essai 7 jours)
- **Chiffrement des données** : toutes les données sont cryptées en localStorage
- **Carte virtuelle** avec QR Code dynamique
- **Gestion complète des adhérents** (CRUD, recherche, filtres)
- **Calcul automatique des arriérés** (tarifs paramétrables)
- **Grand livre comptable** avec historique des versements
- **Module Don de sang** avec filtres par groupe et localité
- **Exports CSV** et **impression** optimisée
- **Verrou de sécurité** avec PIN
- **Gestion des rôles** (Admin, Comptable, Modérateur)
- **Design sombre moderne** et responsive
